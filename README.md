# QR-Auth

QR Code Based 2-Step Authentication

## Installation

Note: Node-qr requires qrencode.

- Mac OS X

    port install qrencode

- Ubuntu / Debian

    apt-get install qrencode
    
- Others

    http://fukuchi.org/works/qrencode/index.en.html
